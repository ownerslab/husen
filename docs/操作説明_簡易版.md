# ClipPad 操作説明（簡易版）

このページは「ZIPにはアプリ本体のみが入っている」前提の最短手順です。

## 1. ZIP を解凍して本体を取り出す

1. `ClipPad-macOS.zip` をダブルクリックして解凍します
2. 解凍して出てくるのは `ClipPad.app` のみです（そのまま次へ進んでください）

## 2. アプリコピー（アプリケーションへ）

1. `ClipPad.app` を Finder で **アプリケーション** フォルダへドラッグします
2. 置き場所は `/Applications/ClipPad.app` を想定します

## 3. 初回起動（ターミナル操作が必要）

初回だけ Gatekeeper 対応のため、隔離属性（quarantine）を解除してから起動します。

ターミナルで、以下を順番に実行してください。

1回目：`xattr -dr com.apple.quarantine "/Applications/ClipPad.app"`

2回目：`open "/Applications/ClipPad.app"`

※ 2回目以降は通常の方法（Finder でダブルクリック等）で起動できます。

## 4. 3つのタブ

| タブ名 | 機能 |
|--------|------|
| **仮置** | コピーした内容の履歴。クリックで前面アプリにペースト |
| **付箋** | 自由記述メモ。自動保存 |
| **辞書** | よく使う語句を登録。クリックで即ペースト |

## 5. 仮置タブの使い方

1. どのアプリでもテキストをコピー（⌘C）すると、一覧に自動追加されます
2. 一覧の項目をクリックすると、青背景にハイライトされ、前面アプリにペーストされます
3. ドラッグで並べ替え、右クリックで削除ができます

## 6. 付箋タブの使い方

1. テキストを自由に入力できます（自動保存）
2. ＋ボタンで新しいメモを追加できます

## 7. 辞書タブの使い方

1. **＋ボタン** で新しい語句を追加（即座に入力モードになります）
2. 語句を入力して **Enterキー** で確定
3. 登録済みの語句を **クリック** すると、前面アプリにペーストされます
4. 語句を編集するには、行をクリックして選択 → **Enterキー** で編集モード → 入力後 **Enterキー** で確定
5. **右クリック** から編集・コピー・削除もできます
