# ClipPad システム概要

## ClipPad とは

ClipPad は macOS 用の「常に最前面」仮置きメモアプリです。Web や他アプリでコピーした**テキスト**を一時的に貯めておき、必要なときに**クリックでクリップボードへ戻して**、他アプリにすぐ貼り付け（⌘V）できる“仮置き場”として使います。

## 想定する使い方（ユースケース）

- Web/資料から複数の文章を拾って、あとでまとめて貼り付けたい
- コピーした断片を「貼り付け順」に並べ替えてから、順番に貼り付けたい
- 常に視界の端に置き、いつでも取り出せる仮メモとして使いたい（複数モニター向け）

## 主要機能（ざっくり）

- **常に最前面表示**: 他アプリの上に表示され続けます
- **自動取り込み**: 他アプリでコピーしたテキストを検知して一覧に追加します
- **クリックで貼り付け準備**: 一覧の項目をクリックするとクリップボードに戻ります（他アプリで ⌘V）
- **並べ替え**: 「編集」からドラッグで順序を入れ替えられます
- **一括削除**: 仮置きデータをまとめて消せます
- **複数モニター対応**: 好きなモニター・位置に置けます

## データの扱い（保存・プライバシー）

- 取り込んだテキストは **端末内**に保存されます（ネット送信はしません）
- クリップボード履歴を扱う性質上、**機密情報のコピー**には注意してください（必要に応じて削除/一括削除）

## 対応環境

- **macOS 13.0 以上**
