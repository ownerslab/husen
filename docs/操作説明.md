# ClipPad 操作説明（ユーザー向け）

## 1. インストール（配布物から使う）

配布された`ClipPad.app`（または`ClipPad-macOS.zip`）を受け取って使う場合の手順です。

### 1.1 ZIP を解凍してアプリを配置

1. `ClipPad-macOS.zip` をダブルクリックして解凍する  
   - ZIP の中にさらに ZIP（`ClipPad-macOS.zip`）が入っている場合は、もう一度解凍してください
2. `ClipPad.app` が出てきたら、Finder の **アプリケーション** フォルダへドラッグして移動する  
   - 例: `/Applications/ClipPad.app`

## 2. 初回起動（Gatekeeper の警告が出る場合がある）

署名なしビルドの配布では、初回起動時に「開けない」「壊れている」等の警告が出ることがあります。  
その場合は、隔離属性（quarantine）を一度だけ解除して起動します。

### 2.1 ターミナルで隔離解除して起動する（初回のみ必須）

配布物（ダウンロードしたフォルダ）に`ClipPad.app`しか入っていない場合、この作業は**ターミナル操作が必要**です。

1. `ClipPad.app` が `/Applications/ClipPad.app` にあることを確認する（アプリケーションフォルダへ移動済みの前提）
2. ターミナルを開いて、次を実行する

```bash
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

※ 初回だけ実行すれば、2回目以降は普通に起動できます。

### 2.2 （参考）リポジトリを持っている場合

ソース一式（このリポジトリ）を手元に置いている場合は、`初回起動用.command`が使えます（上のターミナル操作と同じことを実行します）。

## 3. 2回目以降の起動

- Finder → アプリケーション → `ClipPad` をダブルクリック
- または Spotlight（⌘Space）で `ClipPad` と入力して Enter

## 4. 基本操作（ふだんの使い方）

### 4.1 コピーすると自動で追加される

1. どのアプリでもよいのでテキストを選択して **コピー（⌘C）** する
2. ClipPad の一覧に、そのテキストが追加されます

### 4.2 クリックして貼り付け（いちばん重要）

1. ClipPad の一覧で、貼り付けたい項目をクリックする  
   → その内容が **クリップボードに戻ります**
2. 貼り付け先のアプリへ移動して **貼り付け（⌘V）** する

### 4.3 ウィンドウを移動する

- タイトルバー（上部）をドラッグして好きな位置へ移動できます  
- 複数モニターでも、任意のモニターへドラッグして配置できます

## 5. 一覧の編集（並べ替え・削除）

### 5.1 並べ替え（順番を変える）

1. 画面の **編集** を押す
2. リストをドラッグして順序を入れ替える
3. 編集を終了する（ボタンが戻ります）

### 5.2 1件だけ削除する

- 一覧の項目の操作から「削除」を選びます（右クリック等のメニューがある場合はそこから）

### 5.3 まとめて削除（全消し）

- **一括削除**ボタンで、仮置きデータをまとめて消せます  
  - 誤削除を防ぎたい場合は、削除前に貼り付け先へ退避してください

## 6. 終了方法

- メニューバーの `ClipPad → Quit`、または **⌘Q** で終了できます

## 7. よくあるトラブルと対処

### 7.1 「開発元を確認できないため開けません」「壊れているため開けません」

隔離解除を行います（初回のみ）。

```bash
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

### 7.2 起動時に `_LSOpenURLsWithCompletionHandler error -600` が出る

1. いったん ClipPad を終了（またはプロセスを落とす）
2. Mac を再起動
3. もう一度、隔離解除→起動を試す

```bash
pkill ClipPad 2>/dev/null || true
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

### 7.3 Dock に出ない/出方が想定と違う

アプリの表示ポリシー設定により、Dock 表示が通常アプリと異なる場合があります。  
まずはメニューバーから終了できることを確認してください（`ClipPad → Quit` / ⌘Q）。

## 8. 注意事項（大事）

- ClipPad はクリップボードのテキストを扱うため、**パスワードや個人情報**をコピーした後は必要に応じて削除してください
- 端末内保存のみで完結する設計ですが、画面の見える場所に常時表示する用途のため、周囲から見える環境では内容に注意してください
