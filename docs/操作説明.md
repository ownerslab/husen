# ClipPad 操作説明（ユーザー向け）

## 1. インストール（配布物から使う）

配布された`ClipPad.app`（または`ClipPad-macOS.zip`）を受け取って使う場合の手順です。

### 1.1 ZIP を解凍してアプリを配置

1. `ClipPad-macOS.zip` をダブルクリックして解凍する
   - ZIP の中にさらに ZIP（`ClipPad-macOS.zip`）が入っている場合は、もう一度解凍してください
2. `ClipPad.app` が出てきたら、Finder の **アプリケーション** フォルダへドラッグして移動する
   - 例: `/Applications/ClipPad.app`

## 2. 初回起動（Gatekeeper の警告が出る場合がある）

署名なしビルドの配布では、初回起動時に「開けない」「壊れている」等の警告が出ることがあります。
その場合は、隔離属性（quarantine）を一度だけ解除して起動します。

### 2.1 ターミナルで隔離解除して起動する（初回のみ必須）

配布物（ダウンロードしたフォルダ）に`ClipPad.app`しか入っていない場合、この作業は**ターミナル操作が必要**です。

1. `ClipPad.app` が `/Applications/ClipPad.app` にあることを確認する（アプリケーションフォルダへ移動済みの前提）
2. ターミナルを開いて、次を実行する

```bash
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

※ 初回だけ実行すれば、2回目以降は普通に起動できます。

### 2.2 （参考）リポジトリを持っている場合

ソース一式（このリポジトリ）を手元に置いている場合は、`初回起動用.command`が使えます（上のターミナル操作と同じことを実行します）。

## 3. 2回目以降の起動

- Finder → アプリケーション → `ClipPad` をダブルクリック
- または Spotlight（⌘Space）で `ClipPad` と入力して Enter

## 4. タブ構成

ClipPad には3つのタブがあります。ヘッダー部分のタブ名をクリックして切り替えます。

| タブ名 | 機能 |
|--------|------|
| **仮置** | クリップボード履歴。コピーした内容を一時的に保持する |
| **付箋** | 自由記述のメモスペース。自動保存される |
| **辞書** | よく使う語句を登録し、クリックで即ペーストできるユーザー辞書 |

## 5. 仮置タブ（クリップボード履歴）

### 5.1 コピーすると自動で追加される

1. どのアプリでもよいのでテキストを選択して **コピー（⌘C）** する
2. ClipPad の仮置タブに、そのテキストが追加されます

### 5.2 クリックして貼り付け

1. 仮置タブの一覧で、貼り付けたい項目をクリックする
   → クリックした行が **青背景に白文字** でハイライトされ、内容がクリップボードにコピーされます
2. 貼り付け先のアプリで **貼り付け（⌘V）** する

### 5.3 並べ替え

- リスト内の項目をドラッグして順序を入れ替えられます

### 5.4 削除

- **右クリック → 削除** で1件削除
- ヘッダーの **ゴミ箱アイコン** で全件削除

### 5.5 手動で追加

- ヘッダーの **＋アイコン** をクリックすると、前面アプリで選択中のテキストを取り込みます

## 6. 付箋タブ（メモ）

- 自由にテキストを入力できるメモスペースです
- 入力内容は自動保存されます
- **＋アイコン** で新しいメモを追加、**ゴミ箱アイコン** で全メモ削除

## 7. 辞書タブ（ユーザー辞書）

よく使う語句を登録して、クリックするだけで前面アプリにペーストできる機能です。

### 7.1 語句を登録する

1. **＋アイコン** をクリックすると、新しい行が追加され即座に編集モードになります
2. 語句を入力して **Enterキー** で確定します

### 7.2 登録済みの語句を使う（ペースト）

- 語句の行を **クリック** すると、青背景にハイライトされ、語句がクリップボードにコピーされて前面アプリにペーストされます

### 7.3 語句を編集する

- 編集したい行をクリックして選択（青背景になる）した後、**Enterキー** を押すと編集モードに切り替わります
- 編集後、**Enterキー** で確定します
- または **右クリック → 編集** からも編集モードに入れます

### 7.4 語句を削除する

- **右クリック → 削除** で1件削除
- ヘッダーの **ゴミ箱アイコン** で全件削除

### 7.5 その他の操作（右クリックメニュー）

- **変換語句をペースト** — 語句をクリップボードにコピーし、前面アプリにペースト
- **変換語句をコピー** — 語句をクリップボードにコピーのみ（ペーストはしない）
- **編集** — 編集モードに切り替え
- **削除** — その語句を削除

## 8. ウィンドウ操作

### 8.1 ウィンドウを移動する

- ウィンドウの任意の場所をドラッグして好きな位置へ移動できます
- 複数モニターでも、任意のモニターへドラッグして配置できます

### 8.2 ヘッダーのアイコン

| アイコン | 機能 |
|----------|------|
| ✕ | ウィンドウを非表示（終了はしない） |
| − | ウィンドウを最小化（もう一度押すと元に戻る） |
| ＋ | タブに応じた追加操作 |
| パレット | テーマ（配色）の切り替え |
| ゴミ箱 | タブ内の全データ削除 |

### 8.3 テーマの変更

- パレットアイコンをクリックすると、テーマ選択メニューが表示されます
- 標準 / ダーク / コックピット / パステル系から選べます

## 9. 終了方法

- メニューバーの `ClipPad → Quit`、または **⌘Q** で終了できます

## 10. よくあるトラブルと対処

### 10.1 「開発元を確認できないため開けません」「壊れているため開けません」

隔離解除を行います（初回のみ）。

```bash
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

### 10.2 起動時に `_LSOpenURLsWithCompletionHandler error -600` が出る

1. いったん ClipPad を終了（またはプロセスを落とす）
2. Mac を再起動
3. もう一度、隔離解除→起動を試す

```bash
pkill ClipPad 2>/dev/null || true
xattr -dr com.apple.quarantine "/Applications/ClipPad.app"
open "/Applications/ClipPad.app"
```

### 10.3 Dock に出ない/出方が想定と違う

アプリの表示ポリシー設定により、Dock 表示が通常アプリと異なる場合があります。
まずはメニューバーから終了できることを確認してください（`ClipPad → Quit` / ⌘Q）。

## 11. 注意事項（大事）

- ClipPad はクリップボードのテキストを扱うため、**パスワードや個人情報**をコピーした後は必要に応じて削除してください
- 端末内保存のみで完結する設計ですが、画面の見える場所に常時表示する用途のため、周囲から見える環境では内容に注意してください
